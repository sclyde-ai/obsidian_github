- stateless
    serverがclientの情報を保持しない
- client-server
    clientとserverの役割を明確に分離する
- cacheable
    clientがdataを一時保存する
- uniform interface
    統一されたinterfaceを通じてresourceにaccessする
- layered system
    clientとserverの間に機能ごとに複数の中間層を挟む
    - 中間層の例
        - proxy
        - gateway
        - cache server
